Metashell

You can start an on-line demo of Metashell from this page. It uses shellinabox and in Firefox some keys are not working properly. It is recommended to use other browsers.

Metashell is an interactive shell for template metaprogramming. It is based on the libclang, the Boost.Wave and the Readline libraries.
